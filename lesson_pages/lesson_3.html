The Window Object

"The window object = the browser window that contains a web page

The Browser Object Model (BOM)

• a collection of properties and methods that contain information about the browser and computer screen.
• Every browser window, tab, popup, frame, and iframe has a window object.

* BOM only makes since in a browser environment, not in Node.js.*
* If you don't know the name of the global object, use this:

// from within the global scope
const global = this



• Global variables are actual properties of a global object
• refer to global variables without using the window object



parseInt()

and

isNaN()

are methods of a global object:

window.parseInt(4.2); -- 4 window.isNaN(4.2); -- false

**ES6 made both of these methods of the number object

Dialogs

•

window.alert('Hello');

: will pause the execution of the program and display a message in a dialogue box. the message is provided as an argument to the method, and undefined is returned.

•

window.confirm('Do you wish to continue?')

: program execution is paused and the message provided as an argument displays in a confirmation dialogue with the options of OK or Cancel.

OK = true | | Cancel = false

•

window.prompt('Please enter your name:');

: stops execution, message is the argument. Input field is provided and the text is returned as a string when OK is clicked. Cancel =

null.

Use with care: - these methods with stop the execution of the program.
- use as a final check
- most browsers allow the user to disable dialogs, so they cannot be relied upon.
